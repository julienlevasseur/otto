# Customization

This page documents the [customizations](/docs/appfile/customization.html)
that are available to change the behavior of Java applications with Otto.

## Type: "java"

Example:

```
customization "java" {
    gradle_version = "2.8"
    maven_version = "3.3.9"
}
```
